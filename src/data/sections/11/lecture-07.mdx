## 概要

このレクチャーでは，ReActグラフプロジェクトのボイラープレートセットアップを行います．Poetry環境の構築，依存パッケージのインストール，環境変数の設定を行います．

## セットアップ手順

### 1. Poetryプロジェクトの初期化

```python
poetry init
```

### 2. 依存パッケージのインストール

```python
poetry add langchain langchain-openai langchain-tavily langgraph python-dotenv black isort
```

### 3. 環境変数の設定（.envファイル）

```
OPENAI_API_KEY=your_key
LANGCHAIN_API_KEY=your_key
LANGCHAIN_TRACING_V2=true
LANGCHAIN_PROJECT=react-function-calling
TAVILY_API_KEY=your_key
```

### 4. プロジェクト構造

- `main.py`: エントリーポイント
- `react.py`: 推論エンジン
- `nodes.py`: グラフノードの実装

## まとめ

- Poetryで仮想環境を構築し，LangGraph関連パッケージをインストール
- .envファイルにOpenAI，LangSmith，TavilyのAPIキーを設定
- main.py，react.py，nodes.pyの3ファイル構成でプロジェクトを開始
