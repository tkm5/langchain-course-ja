## 概要

このレクチャーでは，Reflectionエージェントプロジェクトの環境をセットアップします．Poetry環境の構築，依存パッケージのインストール，環境変数の設定を行います．

## セットアップ手順

```mermaid
graph LR
    A["poetry init"] --> B["依存パッケージ<br>インストール"]
    B --> C[".envファイル作成"]
    C --> D["main.py作成"]
    D --> E["動作確認"]
```

### 1. プロジェクト初期化

```bash
mkdir reflection-agent && cd reflection-agent
poetry init
```

### 2. 依存パッケージのインストール

```bash
poetry add python-dotenv black isort langchain langchain-openai langgraph
```

### 3. 環境変数の設定（.envファイル）

```
OPENAI_API_KEY=your_key
LANGCHAIN_API_KEY=your_key
LANGCHAIN_TRACING_V2=true
LANGCHAIN_PROJECT=reflection-agent
```

### 4. ボイラープレートコード

```python
from dotenv import load_dotenv
load_dotenv()

if __name__ == "__main__":
    print("Hello LangGraph")
```

## まとめ

- Poetryで仮想環境を構築し，LangChainとLangGraphをインストール
- GPT-3.5 TurboとLangSmithのAPIキーを設定
- main.pyでボイラープレートコードを作成し動作確認
